# ao2-control-plane v0.1.18 release notes

**Release status:** stable patch release.
**Target tag:** `v0.1.18`

## Patch scope

- Bounds indexed storage reads and paginates storage dashboards so observer
  readback remains predictable as evidence volume grows.
- Adds scalable digest indexing while preserving deterministic, read-only
  storage behavior.
- Hardens release promotion with exact-source post-release baseline binding,
  immutable plan validation, and strict candidate artifact checks.
- Preserves the Control Plane trust boundary: read-only observer surfaces only,
  no AO2 release approval, no provider execution, and no credential changes.

## Archive checksums

Archive checksums are generated from the exact `SHA256SUMS` manifest after the
candidate archives are built. Do not edit archive hashes by hand.
